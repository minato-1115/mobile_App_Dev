### 2. ライブラリのインストール

Flet、pandas をインストールしていただければ、実行環境が整います。

pandasのインストールは次のコマンドを実行してください。

`pip install pandas`

excelファイルを読み込むために次も実行してください。

`pip install xlrd`

Fletのインストールをするために次のコマンドを実行してください。

`pip install flet`

これで実行環境はできました。main.pyを実行すれば、動作します
